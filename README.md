# 커피원두 판매 쇼핑몰

메인페이지에서 회원이 회원가입을 하여 로그인하게 되면 상품을 고른 뒤, 장바구니에 담거나 결제를 할 수 있습니다.
또한 구매한 상품은 마이페이지에서 확인할 수 있습니다.
관리자로 로그인하면 관리자 페이지로 따로 이동할 수 있으며, 관리자 페이지에서는 쇼핑몰의 매출과 모든 상품목록, 주문목록, 회원들의 목록, 게시판글들의 목록을 확인하여 추가/수정/삭제를 할 수 있습니다.
